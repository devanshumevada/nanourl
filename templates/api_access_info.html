How to use the NanoURL Api
  • /api/links - GET : To get all shorted links and related information by a particular user
  • /api/link - POST: To short a link. The request body should be the json data in the form of {'url':URL_TO_BE_SHORTED}
  • /api/link - GET: With a 'short_code' parameter to get related information about the shorted url
  • /api/link?short_code=SHORT_CODE - DELETE: Deletes the short url from the database

All of the requests mentioned above should include Authorization token as 'Authorization':AUTH TOKEN in the request header
